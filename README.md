# 🏛️ Digital Assistant for Legal Literacy

A **conversational AI chatbot** designed to assist users in understanding the **Indian Penal Code (IPC)**. This chatbot utilizes **FAISS for retrieval-based search** and **Hugging Face embeddings** to provide accurate legal information.

---

## 🚀 Features  
- **Natural Language Understanding (NLU)** using `meta-llama/Llama-3.3-70B-Instruct-Turbo` (Together AI API).  
- **FAISS-based Retrieval** for fast and efficient legal information retrieval.  
- **Flask/FastAPI Backend** for quick API responses.  
- **React Frontend** for an interactive chat interface.  
- **Completely Free** (no paid APIs required).
